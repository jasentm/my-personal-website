Read Me

Read Me is an interactive Tarot Card web application that not only teaches you how to read Tarot Cards through multi-faceted quizzes, but also allows users to get a personalized Tarot Card reading, utilizing AI to synthesize the meanings.

Who Wants to Be a Millionaire - CLI

A command line interface (CLI) version of the classic gameshow Who Wants to Be a Millionaire. Users can play the trivia gameshow replete with all 3 lifelines, keep track of their high score, and even add their own questions.

Drinks Galore

A dynamic Single Page Application (SPA) allowing for user input, sorting methods, and graphical models, all revolving around a myriad of drinks. Also included are a hydration tracker that allows for the setting of user goals and a hidden game.
